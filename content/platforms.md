Chef Habitat runs on and supports the following platforms:

- 64 bit Linux kernels 2.6.32 or later
- Modern Linux kernels on 64 bit ARM platforms
- Windows Server 2012 or Windows 8 64-bit or later
- Apple macOS running 64-bit processor running 10.9 or later
